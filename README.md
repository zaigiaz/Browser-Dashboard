# [Browser-Dashboard]
Browser-Dashboard is a startpage that can show weather for any location in the United States, display links for websites, 
and search and read subreddit posts using multiple API endpoints, with the Restful API structure.

# How To Use:

You can clone the repo 
``` 
git clone https://github.com/ZuggRugg/Browser-Dashboard.git
```

This repo contains 3 files (one HTML, CSS and JS file respectively)
You can launch the file by just clicking on the Startpage.htm file and launching it through your browser
You can change the default location of the weather inside the JS file.
