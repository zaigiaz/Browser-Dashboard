[Browser-Dashboard]
---------------------
Browser-Dashboard is a startpage that can show weather for any location in the United States, display links for websites, and search and read subreddit posts using API.



How To Use:
-------------
fork and download the repository. you can run the entire program by clicking and selecting the Startpage.htm file, you can also set the Startpage.htm file as your home page in your browser through settings.
