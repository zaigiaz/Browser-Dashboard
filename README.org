* [Browser-Dashboard]
Browser-Dashboard is a startpage that can show weather for any location in the United States, display links for websites, and search and read subreddit posts using API, with a dynamic background depending on the weather in your location.

It is created only using simple html and javascript code making it lightwieght and efficient.


* How To Use:
fork and download the repository. you can run the entire program by clicking and selecting the Startpage.htm file. Any avaliable web-browser such as Chrome or Firefox should work since it is just a simple html document. You can also set the Startpage.htm file as your home page in your browser through settings.
